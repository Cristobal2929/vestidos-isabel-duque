# Resumen del proyecto

Sitio web de una sola página (SPA) para **Isabel Duque**, boutique especializada en vestidos de ceremonia y eventos.  
Se ha reorganizado la información existente (colecciones, contacto y presentación) en una estructura clara y visualmente memorable, sin usar plantillas ni frameworks.

## Características principales

- **Hero visual** con fotografía de alta calidad que comunica elegancia.  
- Tipografía combinada: **Playfair Display** (títulos) y **Montserrat** (texto).  
- Navegación sticky y accesible, con menú móvil y foco visible.  
- Galerías de imágenes responsivas para cada colección.  
- Cumplimiento de **prefers‑reduced‑motion** y buenas prácticas de accesibilidad.  
- Información de contacto exacta según los datos proporcionados.  

## Tecnologías

- HTML5, CSS3 (variables, grid, flexbox) y JavaScript puro.  
- Google Fonts para tipografía.  
- Sin dependencias externas ni procesos de build.

## Instrucciones de despliegue

1. Copiar la carpeta `assets/` con las imágenes reales (reemplazar los placeholders).  
2. Subir `index.html` y `README.md` al servidor estático o a cualquier hosting de archivos estáticos.  
3. Verificar que los enlaces a redes sociales y al correo funcionen.

## Notas de diseño

- **Color principal**: púrpura (`#6a0dad`) – transmite sofisticación y se asocia a la marca.  
- **Apuesta visual**: la foto del hero, que ocupa toda la pantalla y se repite en la cabecera de cada sección.  
- **Responsive**: el layout se adapta a móviles, tablets y escritorio, manteniendo legibilidad y usabilidad.
